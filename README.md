# PsycSumEval
Corpus of LLM-generated compressed summaries of the abstracts of 115 Cochrane systematic reviews concerning mental health. Summaries were generated by four different LLMs, i.e., GPT-4o, Claude Opus, LLaMA-3.1-70B-Instruct and GPT-4o-mini. yielding 460 summaries in total. Summaries have been evaluated for accuracy and professional acceptability by psychiatrists evaluated each summary across nine content dimensions, by assigning scores and providing free-text justifications that highlight inaccuracies and missing details. 

## Prompt

Summaries were generated by issuing the following prompt to each LLM: 

`Summarize the following research abstract using the PICO framework. Identify the Population (P) being studied, the Intervention (I), the Comparison (C), and Outcomes (O) measured. Highlight the study's methodology, main findings, and conclusions. Emphasize any key clinical outcomes, the quality of the evidence, and how up-to-date the evidence is. Ensure the summary captures the overall significance of the research in the context of patient care and treatment efficacy, as well as any potential limitations or recommendations for future research.`

The prompt was designed to encourage compliance with [PRISMA for Abstracts checklist](https://www.prisma-statement.org/abstracts) and the [Cochrane Handbook guidelines for abstracts](https://www.cochrane.org/authors/handbooks-and-manuals/handbook/current/chapter-iii#section-iii-3-1). Deterministic decoding (temperature <= 0.3) was used when prompting the model. No content beyond the Cochrane abstract was provided to the models. 
## Repository files

### PsycSumEval-Guidelines.pdf

Guidelines followed by the psychiatrists to evaluate each LLM-generated summary


### Cochrane_abstracts.csv

Each row corresponds to a Cochrane systematic review. The columns provide a unique ID for the review, the original abstract provided to the LLMs for summarisation, various bibliographic details, and the summaries generated by each of the four LLMs.

**The current file shows 5 sample rows. The file containing information for all 115 systematic reviews will be made available upon acceptance of the associated manuscript**

### Evaluation_data_{GPT-4o,Claude_Opus,LLaMA-3.1-70B-Instruct,GPT-4o-mini}.csv

Each row corresponds to a Cochrane systematic review. The first column contains the same unique ID for the review provided in Cochrane_abstracts.csv. The remaining columns provide one psychiatrist’s assessments of the summaries generated by the LLM in the file name. The columns provide the information summarised in the table below.

**The current file shows 5 sample rows, i.e., evaluations of the summaries of the abstracts provided in Cochrane_abstracts.csv.  The file containing evaluations of all 115 LLM-generated summaries will be made available upon acceptance of the associated manuscript.**


| Dimension                                                       | Scoring scheme and criteria                                                                                       |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Dimension 1-4: PICO elements (Population, Intervention, Comparator, Outcome) | 0 = missing; 1 = partially correct or inaccurate; 2 = accurate and complete. Reasoning required for 0/1.                          |
| Dimension 5: Quality of Evidence                                           | 1 = consistent with abstract (both present or both absent); 0 = inconsistent/partial information. Reasoning required for 0.            |
| Dimension 6: Main Findings                                                 | 0 = missing; 1 = partial/inaccurate; 2 = accurate. Experts also specify corrective action: add, remove, modify, or none. Reasoning required for 0/1 |
| Dimension 7: Methodology                                                 | Same 0--2 scale and corrective action scheme as Main Findings.                                                                    |
| Dimension 8: Conclusions                                                  | Same 0--2 scale and corrective action scheme as Main Findings.                                                                    |
| Dimension 9: Currency                                                      | 1 = date/status accurate; 0 = inaccurate or omitted; “No date reported” = absent in both abstract and summary.                    |
